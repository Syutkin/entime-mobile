## [Unreleased]
При использовании автоматической отсечки, теперь есть возможность не учитывать от неё данные времени, а использовать вместо этого время получения её на смартфоне. Может быть актуально при сбое/не настроенном времени в отсечке.

### Added
- Реализация стартов через любые промежутки времени
- Настраиваемое временно́е окно старта
- Для автоматических отсечек добавлена возможность использования времени устройства
- Дополнительные настройки темы

## [0.5.0] - 2025-02-03
Полностью переработана организация работы с соревнованиями

### Added
- Добавлена поддержка импорта стартового протокола всего соревнования из csv
- Добавлена возможность добавления этапов из файла в существующее соревнование
- Добавлена возможность добавления трейлов и треков к спецучасткам
- Реализовано автоматическое распознавание кодировки csv файла
- Добавлен список гонщиков и возможность их редактирования
- Добавлен экран только с обратным отсчётом, его можно использовать как стартовое табло для гонщика
- Добавлена возможность использовать звуковое сопровождение без внешнего модуля, используя время на смартфоне
- Для ручных отметок теперь возможно учитывать ntp смещение (при наличии интернета)
- Добавлена индикация разницы стартового и финишного времени между временем модуля и временем устройства
- Добавлен показ всей информации о финишной отсечке
- Добавлен фильтр показа участников в стартовом списке
- Добавлена возможность сдвига времени стартов

### Changed
- Тема приложения использует Material Design 3

### Fixed
- Исправлена ошибка при редактировании стартового времени

### Removed
- Убрана поддержка старых форматов гонок
